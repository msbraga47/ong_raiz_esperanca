# Instituto Raiz da Esperança — Site (Entrega I)

Projeto estático com HTML5 semântico, formulários e assets para a entrega I da disciplina.

## Conteúdo
- `index.html` — página inicial (missão, contato)
- `projetos.html` — lista de projetos e CTA para cadastro/doação
- `cadastro.html` — formulário com validação nativa e máscaras simples
- `css/style.css` — estilo base (mobile-first)
- `js/masks.js` — máscaras simples para CPF, telefone e CEP
- `imagens/*.svg` — imagens em SVG (otimizadas e leves)

## Instruções
1. Extraia o zip.
2. Abra `index.html` no navegador para testar localmente.
3. Para subir no GitHub:
   - Crie um repositório público.
   - Faça `git init`, adicione os arquivos, commit e push para o repositório remoto.
   - Ative o GitHub Pages se quiser um link público (branch `main` ou `gh-pages`).

## Validação W3C
Os arquivos foram construídos para preservar semântica e boas práticas. Recomenda-se que você valide no https://validator.w3.org/ quando adaptar textos ou alterar tags.
